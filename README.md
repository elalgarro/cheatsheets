<h1>Git commit process </h1>
<ul>
  <li>
    git add .
  </li>
  <li>
    git commit -m "message"
  </li>
  <li>
    git push origin HEAD
  </li>
</ul>

<h1> ssh key generator(from reset computer) </h1>
<ul>
  <li>
    mkdir /.ssh
  </li>
  <li>
    ssh-keygen
  </li>
  <li>
    use the public file
  </li>
</ul>
